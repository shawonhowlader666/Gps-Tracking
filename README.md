# GPS Tracking App Source Code Repository

This repository contains the source code for the GPS Tracking applications and admin panel, organized into separate development branches.

## Branches Overview

To access the source code, please switch to the appropriate branch:

- **[onfleet-gps](https://github.com/shawonhowlader666/Gps-Tracking/tree/onfleet-gps)**: Onfleet GPS version of the application. Contains the latest vehicle animations, dynamic course alignment, smooth camera tracking, safe API parsing, and layout updates.
- **[orbit-gps](https://github.com/shawonhowlader666/Gps-Tracking/tree/orbit-gps)**: Orbit GPS version of the application with Orbit branding and assets.
- **[gps-admin-panel](https://github.com/shawonhowlader666/Gps-Tracking/tree/gps-admin-panel)**: PHP Laravel-based GPS Admin Whitelabel Config Panel.
